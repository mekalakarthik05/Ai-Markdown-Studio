Markdown to HTML Converter

Welcome to the Markdown to HTML Converter project.

This tool allows you to convert Markdown files into clean and readable HTML.

---

✨ Features

  • Convert headers
  • Support bold and italic text
  • Handle lists
  • Convert links
  • Render inline code
  • Support code blocks
